# Smart-Energy-Meter-Monitoring-
The Smart Energy Meter Monitoring System is an Electrical Engineering project that monitors electricity consumption in real time. It measures voltage, current, power, and energy usage, helping users track and manage their electricity efficiently. The system aims to reduce energy wastage, lower electricity bills, and promote smart energy management.
#Objectives
To monitor electricity consumption in real time.
To measure voltage, current, power, and energy accurately.
To help users reduce unnecessary electricity usage.
To provide a simple and user-friendly monitoring system.
#Features
⚡ Real-time monitoring of electricity consumption.
📊 Displays voltage, current, power, and energy readings.
📱 Can be connected to a mobile app or web dashboard using IoT (optional).
🔔 Alerts users when power consumption exceeds a set limit.
#Components Used
Arduino uno
ACS712 Current sensor 
Voltage Sensor 
LCD Display 
Breadboard 
Jumper Wires
USB Cable
Power Supply 
Working Principle
The voltage sensor measures the input voltage.
The current sensor measures the current flowing through the load.
The Arduino collects data from both sensors.
The Arduino calculates power and energy consumption.
The measured values are displayed on the LCD screen.
If IoT is used, the data is sent to a mobile app or cloud platform for remote monitoring.
